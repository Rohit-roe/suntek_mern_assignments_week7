# Suntek MERN Assignments - Week 7

This repository contains hands-on React assignments for Week 7 of the Suntek MERN stack training. It consists of three separate React applications built using Vite and Tailwind CSS.

## Projects Overview

### 1. Products Page (`products-page`)
A React application demonstrating component creation and rendering.
- **Concepts:** React components, Props, UI layout.
- **Styling:** Tailwind CSS

### 2. Task Manager App (`Task-Manager-App`)
A robust task management application utilizing forms and state management.
- **Concepts:** React state, Form handling using `react-hook-form`.
- **Styling:** Tailwind CSS

### 3. Country Explorer (`country-explorer`)
An interactive application that fetches and displays real-world data about countries.
- **Concepts:** API integration (`fetch`), `useEffect`, real-time search filtering.
- **Styling:** Tailwind CSS

## Getting Started

To run any of these projects locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rohit-roe/suntek_mern_assignments_week7.git
   cd suntek_mern_assignments_week7
   ```

2. **Navigate to the specific project directory:**
   ```bash
   cd <project-folder-name>
   # Example: cd country-explorer
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
